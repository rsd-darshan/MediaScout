
# MediaScout

Welcome to **MediaScout** – Your Ultimate Hub for Free Stock Videos and Photos! 🚀

## 📸 What is MediaScout?

**MediaScout** is a sleek and easy-to-use platform where you can **explore, search, and download** high-quality **stock videos and photos**. Whether you're a content creator, marketer, or just someone looking for stunning visuals, MediaScout offers you an extensive library of media files to suit your needs.

### 🌟 Why MediaScout?

1. **Vibrant Visuals**: From breathtaking nature shots to dynamic urban clips, find the perfect visual content for your projects.
2. **Seamless Search**: Easily navigate through thousands of videos and images with our intuitive search and filtering system.
3. **User-Friendly Experience**: No complicated UI. Just clean design and simple navigation to help you find what you need, fast!
4. **Dynamic Content**: MediaScout automatically updates with fresh, trending videos and photos so you never run out of inspiration!

## 🔍 Features

- **Explore AI-Powered Agents**: Our AI is here to assist you in finding the right media.
- **Responsive Design**: MediaScout looks stunning on every device – desktop, tablet, or mobile.
- **Pagination for Easy Navigation**: Navigate through content pages smoothly with our pagination feature.
- **Instant Preview**: Hover over images and videos for a quick preview, making your search process even faster.
- **Search by Media Type**: Switch between photos and videos based on what you're looking for.

## 🚀 Getting Started

To run **MediaScout** locally:

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/MediaScout.git
   ```

2. Install the necessary dependencies:

   ```bash
   pip install -r requirements.txt
   ```

3. Run the Flask server:

   ```bash
   flask run
   ```

4. Open your browser and head over to:

   ```
   http://127.0.0.1:5000
   ```

## 💻 Technologies Used

- **Flask** for backend and templating
- **HTML5** and **CSS3** for structure and styling
- **JavaScript** for dynamic interactivity
- **Flexbox** for responsive design
- **Pexels API** (or your stock media source) for fetching media content
